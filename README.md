# 数据通信与网络

一本关于数据通信与网络的教科书。

## 内容结构

```
manuscript/
├── main.tex          # 主文件
├── ch01-*            # 第1章：通信网络基础
├── ch02-*            # 第2章：物理层与数据链路层
├── ch03-*            # 第3章：网络层
├── ch04-*            # 第4章：网络设备与传输优化
├── ch05-*            # 第5章：云网络
├── ch06-*            # 第6章：AI与高性能网络
├── ch07-*            # 第7章：绿色计算
└── ch08-*            # 第8章：移动计算与边缘网络

assets/
├── ch02/             # 第2章图片
├── ch03/             # 第3章图片
├── ch04/             # 第4章图片
├── ch05/             # 第5章图片
├── ch06/             # 第6章图片
└── ...
```

## 编译

```bash
cd manuscript
xelatex main.tex
```
